# Mohammad kinjal — Personal Portfolio Website

A modern personal portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

## About

This portfolio showcases my skills, projects, education, and achievements as a Computer Science student and aspiring Software Developer.

## Tech Stack

* React
* Vite
* Tailwind CSS
* Framer Motion
* React Icons

## Features

* Responsive Design
* Smooth Animations
* Project Showcase
* Skills Section
* Resume Section
* Contact Form

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
